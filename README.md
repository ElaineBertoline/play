# 🎬 Roteiro Play — Creative Director AI
**Transformando links de produtos em roteiros estratégicos com Inteligência Artificial.**

Este projeto foi desenvolvido para automatizar a criação de conteúdo para e-commerce, unindo estratégia de tráfego pago e tecnologia de ponta. 

---

## 🚀 Funcionalidades Técnicas
* **Análise de Produto via IA**: Integração com a API do Gemini 1.5 Flash para ler links e imagens de produtos.
* **Seleção Estratégica Independente**: Interface personalizada que permite escolher simultaneamente o Objetivo (Venda, Autoridade ou Conexão) e a Plataforma (Reels, TikTok, YouTube, etc.).
* **Geração de Roteiro JSON**: A IA entrega uma estrutura de cenas de 8 segundos com ganchos (hooks) e CTAs precisos.
* **Narração Profissional Integrada**: Sistema de síntese de voz (Web Speech API) que permite ouvir, gravar e baixar o áudio de cada cena individualmente.

---

## 🛠️ Tecnologias Utilizadas
* **Linguagens**: HTML5, CSS3 (Design Holográfico/Glassmorphism) e JavaScript Puro.
* **IA**: Google Gemini 1.5 Flash (Google AI Studio).
* **Hospedagem**: HostGator (Subdomínio Privado).
* **Áudio**: Web Speech API & MediaRecorder API.
